# Weather
This is <b>API</b> based Weather Web-App ‚òÅ. The website show weather by searching city names. I have used dynamic icons which changes according to weather of the city.
## API Used
[Open Weather](https://openweathermap.org/current)
  
## API Call
You can call the api by the following methods
1. By City Names
2. By City ID
3. By geographic coordinates
4. By ZIP code
``` js
 api.openweathermap.org/data/2.5/weather?q={city name}&appid={API key}
```
``` js  
api.openweathermap.org/data/2.5/weather?q={city name},{state code}&appid={API key}
```
```js
api.openweathermap.org/data/2.5/weather?q={city name},{state code},{country code}&appid={API key}
 ```
### Example of API Calls
```js
api.openweathermap.org/data/2.5/weather?q=London&appid={API key}
```
```js
api.openweathermap.org/data/2.5/weather?q=London,uk&appid={API key}
```
